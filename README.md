A simple Pomodoro Discord Rich Presence

## Features
- Coming in 2 styles:
  - Listening: Including a progress bar. Aesthetically better.
  - Playing: Legacy way that override your music rich presence. 
- Interactive and straightforward. Just type your session information and begin.

## Requirements
Premodoro uses the `pypresence` library, which can be downloaded from PyPI.
```
pip install pypresence
```

## How to run
### Manually running the Python script
- Clone this repository.
- Run `pomodoro.py`.
```
python /path/to/pomodoro.py
```
or
```
python3 /path/to/pomodoro.py
```
